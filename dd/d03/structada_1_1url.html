Detailed Description

A URL is a struct that represents a universal identifier.

To disambiguate from a valid URL string it can also be referred to as a URL record.

See also
https://url.spec.whatwg.org/#url-representation

◆ non_special_scheme

std::string ada::url::non_special_scheme {}
private

A URL’s scheme is an ASCII string that identifies the type of URL and can be used to dispatch a URL for further processing after parsing. It is initially the empty string. We only set non_special_scheme when the scheme is non-special, otherwise we avoid constructing string.

Special schemes are stored in ada::scheme::details::is_special_list so we typically do not need to store them in each url instance.
